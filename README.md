# 🎮 BouncyBall

경기대학교 | 장희원
Unity 기반 2D/3D 플랫포머 액션 게임 프로젝트



# 🎯 프로젝트 소개

BouncyBall은
플레이어가 공(Ball)을 조작하여 장애물을 피하고 목표 지점에 도달하는
플랫포머 기반 아케이드 게임입니다.

직관적인 조작

점프 및 물리 기반 이동

별 수집 요소

튜토리얼 제공



# 💡게임 방법

| 키 | 설명 |
|----|------|
| ← → | 좌우 이동 |
| Space | 점프 |
| 목표 | 별을 획득하고 클리어 지점 도달 |
